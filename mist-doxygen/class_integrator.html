Detailed Description

Declaration of the Integrator abstract base class, which it the all other integrators inherit from.

This is the interface that all integrators must implement. It provides an abstract Step() function which will be called by the host application to integrate the system forward in time.

Several convenience functions are also provided that are commonly needed by subclasses.
