Detailed Description

A System class which provides accessors for particle data inside the Forcite code.

This is a specialisation of the System interface which is specific to the Forcite code. It provides access to an Integrator to the particle data stored in various arrays within Forcite, and also the ability to compute updates forces, calling Forcite's force evaluation function by means of a callback. This implementation is a stub, the real implementation is property of Dassault Systemes BIOVIA.
