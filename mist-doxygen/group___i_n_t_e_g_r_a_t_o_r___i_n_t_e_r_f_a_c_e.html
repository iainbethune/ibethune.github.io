Get one AMU (Atomic Mass Unit) in internal units.

MIST Integrators work in whatever internal units are used by the host application code. To convert input parameters from a specified unit to internal units, this function returns one AMU (Atomic Mass Unit) (mass).

◆ UpdateForces()

void System::UpdateForces ( )

Update the forces on the system given the current particle positions.

After modifying particle positions, an Integrator may call UpdateForces(). This makes a callback to the host application's force evaluation routine and computes up-to-date forces and potential energies, which may then be read by calling GetForce() / GetPotentialEnergy()
